# Image Processing - Exercise 1: Image Representations and Point Operations

## Overview
This repository contains the first exercise for the Image Processing course. It implements fundamental digital image processing tasks, focusing on image representations, color space transformations, and point operations (intensity transformations). The code leverages efficient, vectorized mathematical operations using NumPy, alongside OpenCV and Matplotlib for reading and visualization.
